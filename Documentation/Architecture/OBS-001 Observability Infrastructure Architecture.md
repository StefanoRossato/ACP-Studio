Abstract

This document defines the architecture of the ACP Studio
Observability Infrastructure.

The Observability Infrastructure provides a unified,
implementation-independent and read-only representation of the
internal behavior of ACP Studio.

Its purpose is to expose system state, events, metrics and
diagnostic information to engineering tools without introducing
dependencies between production components and developer tooling.
