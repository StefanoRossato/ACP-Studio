# ACP Studio

ACP Studio is an engineering framework for REAPER designed to prepare
projects for professional analog mixing.

The framework combines Lua, JSFX, and a deterministic runtime
architecture to provide a reliable, modular, and extensible platform
for audio analysis and analog workflow automation.

---

# Mission

Provide an engineering platform for preparing REAPER projects for
professional analog mixing.

---

# Technology

ACP Studio is built using:

- REAPER
- Lua 5.4
- JSFX
- Shared Memory IPC

# Repository Philosophy

ACP Studio is developed incrementally.

Every milestone:

- implements one engineering capability;
- follows an approved specification;
- is experimentally validated inside REAPER;
- completes Architecture Review;
- completes Repository Cleanup;
- is certified;
- updates the CHANGELOG;
- ends with a Git commit;
- is identified by a milestone tag.

Engineering quality always takes precedence over feature growth.
